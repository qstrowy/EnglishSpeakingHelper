How to use this practice
  1. Draw a topic
  2. Use prep time to think of keywords, not full sentences
  3. Speak for 1 minute
  4. Notice where you got stuck
  5. Repeat the same topic and speak again more clearly (here the real learning happens!)

The goal is not to write a perfect answer. The goal is to organize your thoughts and speak more clearly each time.
